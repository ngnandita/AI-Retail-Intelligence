# 🛒 AI Retail Intelligence

## Overview

AI Retail Intelligence is a Machine Learning and Data Analytics project built using Python and Streamlit.

It predicts retail sales and provides business insights through an interactive dashboard.

---

## Features

- Sales Prediction using XGBoost
- Business Dashboard
- KPI Cards
- Sales by Category
- Monthly Sales Trend
- Monthly Profit Trend
- Top Customers
- Business Insights
- CSV Upload
- Download Business Report

---

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- XGBoost

---

## Dataset

Superstore Sales Dataset
